Module 1: Introduction to Data Structures
Overview of Data Structures.
Scope and core use cases for common structures.
Common array issues.
Bounds, resizing, and time complexity pitfalls.
Revision on Dynamic Allocations in C/C++.
Memory allocation, pointers, and lifetime review.
Module 2-3: Linked Lists
Implementation of singly Linked Lists (insertion, deletion, traversal).
Build and traverse single-linked nodes.
Concepts of doubly and circular linked lists.
Variants, trade-offs, and use cases.
Tortoise and Hare (Floyd's) cycle algorithm.
Two-pointer cycle detection technique.
Module 4-5: Abstract Data Types - Stacks & Queues
Introduction to Abstract Data Types (ADT)
Define operations independent of implementation.
Implementation of stacks and queues using arrays.
Array-backed stack/queue operations.
Implementation of stacks and queues using linked lists.
Linked-list stack/queue operations.
Stack application: Postfix expression conversion and evaluation (Problem solving).
Use stacks for expression evaluation.
Module 6, 8-9: Trees
Introduction to trees: Binary trees and Binary search trees (BST)
Tree terminology and BST properties.
Tree traversal algorithms: Depth first and Breadth first.
DFS vs BFS traversal patterns.
Tree operations: Insertion, deletion, searching.
Core BST operations and costs.
Balanced Trees: AVL rotations OR Red-Black trees.
Balancing to control height.
Module 10: Advanced Tree Structures
Huffman Coding and Huffman trees.
Prefix codes and optimal trees.
Priority Queues.
Priority-based insertion and removal.
Course Project Announcement.
Project scope and milestones.
Module 11, 13: Hash Tables and Heaps
Introduction to Hashing.
Hash functions and table basics.
Hash table implementation using arrays.
Array-based hash table mechanics.
Explanation of Heaps and their basic functions.
Heap properties and operations.
Collision resolution techniques: Chaining and Open addressing.
Resolve collisions with chaining or probing.
Module 14-15: Graphs
Introduction to Graphs: Representation and terminology (adjacency matrix and adjacency list).
Graph models and storage methods.
Minimum Spanning Trees (Prim's and Kruskal's algorithms)
MST algorithms and use cases.
Shortest Distance/Path algorithms (Dijkstra's algorithm)
Shortest path fundamentals.
